# 19 Post-MVP Roadmap

## Phase 2: Etsy draft publishing

Build Etsy OAuth and draft listing creation.

Features:

- Connect Etsy shop.
- Create local listing draft.
- Push draft to Etsy.
- Upload listing images.
- Upload digital ZIP files.
- Set listing type to download.
- Leave listing inactive/draft for seller review.

Do not auto-publish by default.

## Phase 3: Advanced upscaling

Add `UpscaleProvider` implementations:

- high-quality art upscaler,
- line-art upscaler,
- photo/art mode selection,
- per-ratio advanced upscale.

Add before/after zoom comparison.

## Phase 4: AI outpaint for ratio conversion

When crop would cut important content:

- extend canvas,
- use image edit/outpaint,
- preserve central subject,
- let user choose between crop and outpaint.

## Phase 5: Batch collections

Generate cohesive sets:

- 3-piece gallery wall sets,
- 6-pack nursery collection,
- seasonal drops,
- style-consistent series.

Needs consistency controls and stricter review.

## Phase 6: Shop style memory

Let seller save:

- brand palette,
- preferred styles,
- default disclosure,
- default description template,
- banned themes,
- recurring tags.

## Phase 7: SEO assistance

Add:

- keyword planner,
- competitor-free generic inspiration research,
- title/tag variants,
- listing score checklist.

Do not scrape Etsy in ways that violate terms.

## Phase 8: Mockup library

Add high-quality deterministic mockups:

- living room,
- nursery,
- bedroom,
- office,
- frame grid,
- seasonal mockups.

Allow seller to upload own mockup backgrounds if they have rights.

## Phase 9: Print-on-demand bridge

Integrate print providers for physical products after digital flow works:

- Printful/Gelato/Printify variants.
- Product templates.
- Production partner disclosure helper.

This is a separate product line; do not mix into MVP.

## Phase 10: Team/workspace features

- Multiple shops.
- Collaborators.
- Approval workflow.
- Shared presets.
- Team credit pool.

## Phase 11: Marketplace of presets

Allow curated style packs, but avoid IP-risky styles.

## Phase 12: Custom model/style training

Potentially allow sellers to train on their own licensed artworks.
Needs strong rights confirmation and data handling.
